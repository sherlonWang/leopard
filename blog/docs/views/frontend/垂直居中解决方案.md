css 布局垂直居中使用弹性布局：

```html
<div class="father">
  <div class="son">
    
  </div>
</div>
<style>
  .father {
     display: flex;
     align-items: center;
     justify-content: center;
  }
  .son {
    
  }
</style>
```

父节点使用弹性布局：

```css
display: flex;
align-items: center;
justify-content: center;
```
